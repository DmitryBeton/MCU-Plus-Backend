# MCU-Plus-Backend
серверная часть проекта для управления расписанием учебных занятий и предоставления данных мобильному приложению MCU Plus

## Локальный запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Сайт будет доступен по адресу http://127.0.0.1:8000/.

## Импорт расписания

На странице расписания можно загрузить JSON-файл или Excel-файл с готовыми занятиями. Для JSON поддерживаются два варианта:

- плоский список в поле `entries`;
- вложенная структура `institutes -> groups -> lessons`.

Пример вложенного файла лежит в `docs/schedule-import-example.json`.

Для Excel поддерживается формат учебного отдела, где:

- название листа содержит курс и группу, например `2к ЦИБ-241`;
- строки `день / неделя/дата` начинают блок дня недели;
- следующая строка содержит даты по неделям;
- колонка `время` содержит время пары;
- ячейки на пересечении даты и времени содержат описание занятия.

## API для мобильного приложения

Основной API для iOS:

```text
GET /api/v1/catalog/
GET /api/v1/schedule/?groupId=1&from=2026-02-09&to=2026-02-16
```

Старые технические endpoint-ы для отладки:

```text
GET /api/groups/
GET /api/schedule/?group_id=1
```
